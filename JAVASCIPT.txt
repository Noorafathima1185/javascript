JAVASCIPT
---------
- can be used in both front-end as well as back-end
- front-end : behavior
- back-end : logical operation

- single thread programming language - line by line execution 
- javascript is dynamically typed language (data type changes)

- node.js - its an open source javascript run time environment, that execute javascript code outside web browser.

- extension - filename.js

display
-------
console.log(content)
console.log('content',variable)

data types
----------
primitive
1) string
2) number
3) boolean - true or false
4) bigInt - number larger than 2^53 and numbers less than -2^53
5) undefined - variable declared - value are not assigned let x
6) null - let x=null
7) symbol - unique identifier

non primitive - combination of primitive datatype
8) object - values are stored has key value pair 

variables
---------
- used to store data

identifiers - rules to create variables
-----------
1) it can be started with alphabets ,dollar symbol or underscore
2) first character can be followed by alphabets ,dollar symbol , underscore and digits
3) digit cannnot be placed as the first character
4) can use more than one  word but the words should have place
    - second word can start with capital letter(camel case)
    - use underscore to separate the word

syntax
------

variable_name = value

keyword variable_name = value

keyword : reserved words are called keywords which gives a special meaning to the variables
1)const - reassigning not possible ,  hoisting not possible  -functional scope
2)var   - reassigning possible ,      hoisting possible      -functional scope
3)let   - reassigning possible ,      hoisting not possible  -block level scope ({})

variable hoisting - using a variable before it is declared

4)concatination - adding 2 datatypes
---------------
number + number = number
string + string = string
string + number = string

5)Template Literals
-------------------
another way of displaying content
`content ${variable_name}`


6)operators - boolean value
-----------
   1)Assignement operator ( = ) 
   2)Arithmatic operator ( + , - , / , * , ** , % )
   3)Relational operator ( > , >= , < , <= , == , ====)

   == value (left value is same as right value) eg:- 10 =='10'   (true)      comparison
   === check the value & datatype eg:- 10==='10'  (false)
